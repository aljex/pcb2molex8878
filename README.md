# pcb2molex8878 - "max" version
PCB and 3d-printable model to fit in the Molex 8878 DIP-28 socket found in some vintage computers.
![](pcb2molex8878.jpg)

This variant maximizes the usable pcb real-estate, on both top & bottom surfaces.
